# App Icons

- `16x16 Assorted RPG Icons` by Shade: https://opengameart.org/content/16x16-assorted-rpg-icons
- `Game icons` by Kenney: https://opengameart.org/content/game-icons

The source pages list these assets as Creative Commons Zero (CC0). Kenney credit is appreciated but not required.
