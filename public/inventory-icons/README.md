# Inventory Icons

Organized into subdirectories by type:

- `plants/` — Icons for all 99 crops in the plants collection (matched by crop_id)
- `animals/` — Icons for all 8 livestock in the livestock collection (matched by livestock_id)
- `materials/` — Icons for tools and materials

## Sources

- `16x16 Food` by ARoachIFoundOnMyPillow: https://opengameart.org/content/16x16-food
- `CC0 Tool Icons` collection by AntumDeluge, using `tool-yd.zip`: https://opengameart.org/content/cc0-tool-icons

Both source pages list the license as Creative Commons Zero (CC0).

Plants without a matching icon in the food pack have generated 16x16 pixel art placeholders
using distinct colors per plant type.
